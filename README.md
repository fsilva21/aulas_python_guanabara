# Exercícios de Python - Curso em Vídeo

## Sobre o repositório

Esse repositório contém **todos os 115 exercícios resolvidos** do curso de Python do professor **Gustavo Guanabara**.  
Organizei tudo em pastas separadas por módulos: 'Mundo 1', 'Mundo 2' e 'Mundo 3'.

---

## Estrutura do Curso

### Mundo 1 (Exercícios 001 ao 035)
- **História do Python e Seus Usos**
- **Variáveis**
- **_F-Strings_**
- **Operações Aritméticas e Relacionais**
- **Estruturas Condicionais Utilizando `if/else`**
- **Importação de Módulos/Bibliotecas**
- **Tratamentos Iniciais em _Strings_**

### Mundo 2 (Exercícios 036 ao 071)
- **Condições Aninhadas com `if/elif/else`**
- **Laços de Repetição `for` e `while`**
- **Laço Infinito com `while True` e `break`**

### Mundo 3 (Exercícios 072 ao 115)
- **Tuplas, Listas e Dicionários**
- **Funções `def`: Parâmetros Opcionais, _docstrings_ e Escopo Local/Global**
- **Ajuda Interativa com o `help()`**
- **Criação de Pacotes e Módulos Personalizados**
- **Tratamento de Erros com `try`, `except`, `finally`**
- **Projeto Final: Sistema Interativo com Acesso a Arquivos de Texto**

## Como Executar os Arquivos .py

1. Certifique-se de ter o **Python** instalado ou atualize-o para a última versão.
2. Abra o terminal na pasta onde o arquivo se encontra ou use o PyCharm.
3. Execute o arquivo desejado digitando a seguinte linha de comando:

**Linux**
```bash
python3 ex012.py 
```
**Windows**
```bash
python ex012.py 
```
